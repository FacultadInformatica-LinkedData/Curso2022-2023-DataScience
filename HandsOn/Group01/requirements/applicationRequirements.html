Application Requerimets

Requerimets for the application of the data in the project.

In this project, the application given to the data is to create relationships between libraries and public schools in Madrid. One example of it could be the distance between two institutions of the data base. Others applications of the selected data is: to get the easiest way to go to any place in the data base by using public transport, see and compare the equipment of the schools and libraries, compare the timetables of the libraries or study the descriptions of the datasets: the type of school in case of the first dataset and the surface and other staff in case of the second dataset.

In this line, the user interface is focus in the searchers of the institutions, and when one of them is selected, you will choose between the differents options of data related to your choice. For example, if you choose a school you have the option to quest the type, place, name of the street where is located, etc.
