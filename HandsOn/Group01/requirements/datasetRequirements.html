Dataset requirements

R1: Data are in one of the selected smart city domains.

The dataset contains information about traffic accidents involving bicycles. This information has been collected and published by the Madrid City Council on its Open Data Portal.

R2: Data are available as a CSV file.

The dataset is available in CSV format.

R3: Data have an open license so they can be published.

The open data portal allows the reuse of this and other data sets for commercial and non-commercial purposes. For more information, you can consult the Conditions of Use.

R4: Data can be easily linked with generic real-world entities (e.g., locations).

The data set includes the exact location where the accident occurred (coordinates, district and street), as well as the data of the driver involved and his positive/negative in the alcohol consumption test.

R5: Documentation exists for the data (optional).

PDF documentation appears on the same page where the dataset was extracted. This documentation includes a description of the fields that appear in the dataset

R6. Data can come from multiple data sources (optional).

In this case, the main source of data is the Madrid Municipal Police, although the City Council also supports the veracity of this data. In addition, the portal also includes the dataset of police actions, as a way to complement this data.
