Name strategy

Las uris se agruparán en dos categ.

  • Uris para definir la ontología
  • Uris para definir los recursos inicializados

Uris de la Ontología

Todas ellas comienzan por: https://www.biblioescuela.educamadrid.org/ontologies/
Este tipo de Uris quedarán divididas en:

  • Uris para definir la Clase
  • Uris para definir la Propiedad

Las Uris de definición de clases para "Library" tendrán el siguiente formato: https://www.biblioescuela.educamadrid.org/ontologies/Library#[class_name]
Las Uris de definición de propiedades para "Library" tienen el siguiente formato: https://www.biblioescuela.educamadrid.org/ontologies/Library#[property_name]
Las Uris de definición de clases para "School" tendrán el siguiente formato: https://www.biblioescuela.educamadrid.org/ontologies/School#[class_name]
Las Uris de definición de propiedades para "School" tienen el siguiente formato: https://www.biblioescuela.educamadrid.org/ontologies/School#[property_name]

Uris de los Recursos

Todas ellas comienzan por: h​ttps://www.biblioescuela.educamadrid.org/resource/
Las Uris se dividiran en dos tipos:

  • Uris referentes a Library

En esta categoria: las Uris tienen esta forma:
 https://www.biblioescuela.educamadrid.org/resource/Library/[class_name]/[identifier]

  • Uris referentes a School

En esta categoria: las uris tienen esta forma:
https://www.biblioescuela.educamadrid.org/resource/School/[class_name]/[identifier]
